linux上 redis的安装

2021-11-09 1 min read

前言

redis服务器我曾经安装过好几次，虽然很简单，但是每次都需要查询资料后才能顺利完成，这样的处事方法效率实在是太低了，正好这次我又要安装redis,所以我决定将安装的过程记录下来，一方面可以增强自己的记忆，另一方面也能方便我随时查阅

下载

前往redis下载页面，选择稳定版本进行下载
或者直接命令行

wget http://download.redis.io/releases/redis-5.0.3.tar.gz

解压

tar -zxvf redis-5.0.3.tar.gz

编译

cd redis-5.0.3/

make

可能遇到的错误

You need tcl 8.5 or newer in order to run the Redis test

解决方法

安装tcl

yum install tcl

安装

make install

启动

cd /usr/local/bin

redis-server redis.conf
