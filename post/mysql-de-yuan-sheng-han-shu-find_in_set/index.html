Mysql的原生函数find_in_set

2021-11-09 1 min read

find_in_set是mysql中的一个原生函数

在日常开发中，经常会用到in 函数进行指定序列的查询，但是有时候需要进行反向的业务查询，这时候就用到了find_in_set函数。

用法

SELECT FIND_IN_SET('b', 'a,b,c,d')

在实际应用中，可以把b替换成传入的参数，后面的序列替换成数据库的字段，该字段的值必须是以英文逗号分隔的字符串
